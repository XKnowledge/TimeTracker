# 时光刻录 (TimeTracker)

一款简洁优雅的个人时间追踪桌面应用，帮助你记录每日活动、追踪时间分配、分析时间使用效率。

## 功能特性

### 核心功能
- **每日时间追踪** - 设置每日起始时间作为计算基准，添加事件并自动计算时间间隔
- **事件管理** - 添加、删除、编辑事件，支持标记重要事项
- **统计仪表盘** - 可视化展示总时间、重要事件时间及占比

### 数据管理
- **日期导航** - 支持日期选择器、前后日切换、快速返回今天
- **数据持久化** - 自动保存到本地 JSON 文件
- **导入/导出** - 支持数据备份和恢复

### 用户体验
- **现代 UI 设计** - 简洁美观的界面，带有渐变装饰效果
- **响应式布局** - 适配桌面和移动设备
- **流畅动画** - 淡入、滑入等过渡动画效果
- **Toast 通知** - 操作反馈提示

## 安装与使用

### 环境要求
- Node.js 16+
- npm 或 yarn

### 快速开始

```bash
# 克隆仓库
git clone https://github.com/your-username/TimeTracker.git
cd TimeTracker

# 安装依赖
npm install

# 启动应用
npm start
```

### 使用说明

1. **设置起始时间** - 调整时间选择器设置每日开始时间（默认 08:00）
2. **添加事件** - 点击"添加事件"按钮创建新条目
3. **填写详情** - 输入事件描述、设置完成时间、可标记为"重要"
4. **查看统计** - 在统计卡片中查看时间分配情况
5. **切换日期** - 使用日期选择器或箭头按钮查看/编辑其他日期
6. **备份数据** - 使用导出按钮保存 JSON 备份
7. **恢复数据** - 使用导入按钮加载备份文件

## 构建打包

```bash
# 构建 Windows 版本
npm run build:win

# 构建 macOS 版本
npm run build:mac

# 构建 Linux 版本
npm run build:linux
```

## 数据结构

数据以 JSON 格式存储在 `timetracker-data.json` 文件中：

```json
{
  "2026-02-22": {
    "startTime": "08:00",
    "events": [
      {
        "id": 1771694126689,
        "important": false,
        "description": "事件描述",
        "time": "08:20"
      }
    ]
  }
}
```
